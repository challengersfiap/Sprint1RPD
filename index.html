Descrição do projeto

O objetivo da nossa solução é integrar um software de reconhecimento de imagem capaz de reconhecer se as imagens contém uma bicicleta do modelo que o usuário disse possuir e se ela está em boa condição, juntamente com isso haverá o uso de instruções detalhadas para o usuário para garantir que as fotos serão de qualidade e integração com banco de dados para armazenamento, e um chatbot para auxiliar com dúvidas que possam surgir no meio do processo. A pagina de Fotos é apenas para demonstrar a ordem dos processos na aplicação, pois ela utilizará a camera do aparelho do usuario para tirar as fotos no momento da vistoria, diminuindo as chances de ocorrerem fraudes.
